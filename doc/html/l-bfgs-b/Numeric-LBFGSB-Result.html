Solution point if the minimization completed successfully. See stopReason.

backtrace :: [Vector Double]

The steps taken to reach the solution, in reverse order. Does not include the starting point.

stopReason :: StopReason

The reason L-BFGS-B terminated. Only if this is Converged should you consider the solution correct!
